# 申请白名单

  1. 加入QQ群597691030 (opens new window)或Discord群组 (opens new window)
  2. 修改你的群名片或昵称为你的Minecraft ID
  3. 在在线文档-成员数据中填写信息（此步骤待完善）
  4. 在群内发送#绑定：你的ID
  5. 在服务器官网阅读服务器规定
  6. 找服主(Glucy2 (opens new window))或服管理(na2hco3_CN (opens new window)或EDGW_ (opens new window))添加白名单
